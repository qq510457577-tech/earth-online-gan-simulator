# GAN半球穹顶防御系统 - 项目记忆

## 项目概述
- 3D伪立体半球穹顶攻防对抗系统
- 攻方: 30个单元，在半球表面随机移动、饱和发射小球
- 守方: 单台红外智能锁定，支持串口控制云台
- GAN自学习进化

## 功能模块

### 1. 半球3D伪立体穹顶渲染
- 多层等高线 (5层)
- 经线 (18条)
- 地平线 + 内部圆环
- 顶点光晕效果

### 2. 攻方集群
- 30个红色攻击单元在半球表面随机移动
- GAN生成发射策略 (angle, elevation, launch_prob)
- 饱和发射控制
- **红色小球彗星尾巴效果** (2026-03-26升级)

### 3. 守方多目标优先级锁定
- 三种模式: nearest(最近) / fastest(最快) / threat(威胁度)
- compute_threat() 计算威胁值

### 4. 串口云台控制
- 输出格式: PAN <角度> TILT <角度> LASER <1/0>
- 支持COM3/9600波特率

### 5. GAN模块
- Generator: 16维输入 -> N*3维输出 (移动+发射)
- Discriminator: 6维特征 -> 命中概率
- **支持动态学习率调整** (2026-03-26升级)

### 6. 对抗日志 + 实时曲线图
- 命中/发射/命中率/GAN损失 实时记录
- matplotlib后台渲染 -> pygame surface

### 7. GUI参数调节面板 (2026-03-26大升级)
| 滑块 | 参数名 | 范围 |
|------|--------|------|
| 1 | AttackerN (攻方数量) | 10-50 |
| 2 | LaunchProb (发射概率) | 0.05-0.6 |
| 3 | MaxBalls (最大小球) | 20-200 |
| 4 | BallSpeed (小球速度) | 0.5-6.0 |
| 5 | IRRadius (红外半径) | 6-40 |
| 6 | DomeTilt (穹顶倾斜) | 0.1-0.9 |
| 7 | TurretSpd (云台速度) | 1.0-15.0 |
| 8 | DomeRadius (半球半径) | 150-350 |
| 9 | GenLR (Generator学习率) | 0.0001-0.005 |
| 10 | DiscLR (Discriminator学习率) | 0.0001-0.005 |
| 11 | SimSpeed (模拟速度) | 0.1-3.0x |

优先级按钮: 最近/最快/威胁

### 8. 动画特效 (2026-03-26升级)
- **红色小球彗星尾巴**: 渐变大小和透明度
- **黄色激光轨迹**: 渐隐效果 + 头部亮点
- **击中爆炸特效**: 橙色光芒扩散动画

## 代码结构
- `gan_dome_defense.py`: 主程序 (~700行)
- 依赖: pygame, torch, numpy, matplotlib, pyserial

## 运行
```bash
pip install pygame torch numpy matplotlib pyserial
python gan_dome_defense.py
```

## 操作说明
- ESC - 退出程序
- 鼠标拖动滑块实时调节参数
- 点击按钮切换锁定模式
- 串口按钮开启/关闭硬件控制
